# Dairy Pro - Mobile App

Welcome to the **Dairy Pro Mobile App** repository! This is the mobile application component for the Dairy Pro platform.

## Features
- Complete cattle management and tracking.
- Health monitoring and vaccination schedules.
- Milk production tracking and analytics.
- Inventory, HR, and Finance management.
- Subscription and settings management.

## Tech Stack
- React Native / Expo
- Tailwind CSS
- Firebase

## Getting Started

1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Start the application with `npx expo start`.
